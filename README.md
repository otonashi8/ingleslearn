# 🇺🇸 EnglishLearn - Tutor de Inglés por Voz con IA

EnglishLearn es un asistente interactivo que te ayuda a practicar inglés mediante conversaciones por voz con IA.

## 🎯 Características

- 🎤 **Interacción por voz** - Presiona el botón y habla
- 🤖 **IA avanzada** - Usa Groq + LLaMA 3.1
- 📝 **Traducción instantánea** - Cada respuesta en inglés y español
- 💡 **Sugerencias** - 4 opciones para continuar la conversación
- ⏰ **Quiz automático** - Cada 5 minutos sin hablar

## 🌐 Demo Online

## 🚀 Tecnologías

- **Frontend:** HTML5, CSS3, JavaScript (Web Speech API)
- **Backend:** Python, Flask, Groq API, Google TTS
- **Despliegue:** GitHub Pages + Render
